This APK was built without a server URL. To use OpenCode, rebuild the APK pointing at your self-hosted instance:

OPENCODE_SERVER_URL=https://your-server.example.com pnpm build:android
1. Deploy the OpenCode server (Docker or VPS)
2. Set OPENCODE_SERVER_URL to its address
3. Run pnpm build:android and rebuild the APK
4. Open android/ in Android Studio → Generate Signed APK
